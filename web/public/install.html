The Project

This project provides a comparison of HTML/JavaScript to the Microsoft Windows Presentation Foundation (WPF) in desktop applications. The HTML versions utilize the packages scalejs and windowmanagerjs and can be ran on the desktop inside the containers OpenFin or Electron or it can be viewed inside any browser - all with +99% code reuse. The WPF version utilizes a fork of the CodePlex package FastWPFGrid.
